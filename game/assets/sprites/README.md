# Sprite Assets

将以下名称的 PNG 图片放入此文件夹，程序会自动加载并替换默认的彩色占位方块：

| 文件名 | 用途 | 建议尺寸 |
|---|---|---|
| `wall.png` | 墙壁 | 任意（自动缩放到格子大小） |
| `floor.png` | 地板/通道 | 任意 |
| `player.png` | 玩家角色 | 任意 |
| `coin.png` | 金币 | 任意 |
| `trap.png` | 陷阱 | 任意 |
| `boss.png` | Boss | 任意 |
| `start_portal.png` | 起点传送门 | 任意 |
| `end_portal.png` | 终点传送门 | 任意 |
| `ai_marker.png` | AI 标记 | 任意 |

**注意事项：**
- 支持透明背景（PNG alpha 通道）
- 图片会被自动缩放到当前格子大小（程序根据迷宫尺寸自动计算）
- 缺少任何文件时，程序会使用彩色方块+文字标签作为占位图
- 建议使用正方形图片以获得最佳效果
