# Sound Assets

将以下名称的音频文件放入此文件夹，程序会自动加载并播放：

| 文件名 | 用途 |
|---|---|
| `step.wav` / `step.ogg` | 移动音效 |
| `coin.wav` / `coin.ogg` | 收集金币音效 |
| `trap.wav` / `trap.ogg` | 触发陷阱音效 |
| `boss_start.wav` / `boss_start.ogg` | Boss战开始 |
| `boss_hit.wav` / `boss_hit.ogg` | 攻击音效 |
| `boss_defeat.wav` / `boss_defeat.ogg` | Boss击杀 |
| `victory.wav` / `victory.ogg` | 胜利 |
| `defeat.wav` / `defeat.ogg` | 失败 |
| `ui_click.wav` / `ui_click.ogg` | UI 点击 |

**注意事项：**
- 支持 `.wav` 和 `.ogg` 格式（优先 `.wav`）
- 缺少任何文件时，对应音效将被静默跳过，不会报错
