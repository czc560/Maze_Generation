# 答辩展示提纲

1. 项目目标：设计可验证、可视化、可挑战 AI 的竞技迷宫。
2. 迷宫规则：符号、连通性、唯一 S/E/B、资源可达。
3. 五种生成算法演示：DFS、Prim、Kruskal、Division、BFS Optimize。
4. 验证器与指标：完美迷宫、死胡同、分支、路径长度。
5. 资源放置与最优资源路径。
6. BOSS 战：技能冷却、分支限界、最少回合。
7. AI 玩家：Greedy、PPO、Q-learning、自定义策略。
8. pygame 界面：开始画面、加载动画、路径闪烁、游戏 HUD、BOSS 画面接口。
9. 素材与音效扩展：AssetManager、AnimationManager、AudioManager。
10. 实验输出：JSON、图片、AI 路径、结果摘要。
